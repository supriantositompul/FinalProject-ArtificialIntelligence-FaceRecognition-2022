# [UTS] - Linkset

## Clone the project
```bash
git clone https://github.com/itdel-ppw/22-03-linkset.git
```

File ```public``` dibuat dalam bentuk zip sehingga harus dilakukan ekstrak terlebih dahulu
## Extract Public Folder and Go to the project directory
```bash
cd webapp
```
```bash
composer install or composer update
```
```bash
php artisan key:generate
```
Customize database from MySQL to .env
```bash
php artisan migrate
```
```bash
php artisan serve
```
